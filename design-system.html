Motion

Never use linear or ease. All transitions must use one of these three curves. Always include a prefers-reduced-motion override.

Token / Name Curve Timing Use when
UI Response
--bx-transition		 cubic-bezier(0.25, 0, 0, 1) 100–200ms Buttons, hover states, color transitions
Delightful cubic-bezier(0.34, 1.56, 0.64, 1) 200–350ms Card entry, stagger animations, celebrations
Exit cubic-bezier(0.4, 0, 1, 1) 150–250ms Elements leaving the screen, fade-outs
Page transition cubic-bezier(0.25, 0, 0, 1) 200–350ms Page/route changes
Press / Active cubic-bezier(0.25, 0, 0, 1) 100ms scale(0.97) on button:active
/* Stagger: 30–50ms per item, max 300ms total */ .feed-item:nth-child(1) { animation-delay: 0ms; } .feed-item:nth-child(2) { animation-delay: 50ms; } .feed-item:nth-child(3) { animation-delay: 100ms; } /* Always required */ @media (prefers-reduced-motion: reduce) { * { animation-duration: 0.01ms !important; transition-duration: 0.01ms !important; } }

Z-index Scale

Layer ordering สำหรับทุก element ใน BirdieX. Developer ต้องใช้ค่าจาก table นี้เท่านั้น ห้ามใช้ค่า arbitrary เช่น z-index: 9999.

Layer Name z-index Used by Notes
Base / Background 0 #bg-illustration #match-bg-illustration Golf course background image ที่อยู่ใต้ทุกอย่าง
Content 1 .scroll-body .card .feed-item #app::before (gradient) Content cards อยู่เหนือ background gradient
Elevated Content 2–3 .match-header elements .tab-bar .match-title Content เหนือ header watermark overlay (z-index 1)
Sticky Tab Bar 50 .profile-tabs (sticky) Tab bar ติด sticky ด้านบน scroll เนื้อหาโปรไฟล์
Fixed UI 99 #bottom-nav .fab Navigation และ FAB — ลอยอยู่เหนือ content เสมอ
Top Chrome 100 .header .match-header (z-index: 100 via position) Status bar + top nav — highest permanent layer
⚠️ Never use 999 / 9999 — ห้ามใช้เด็ดขาด — ใช้ค่าจากตารางข้างต้นเท่านั้น เพื่อป้องกัน stacking conflict
/* Z-index reference — ใช้ค่าเหล่านี้เท่านั้น */ --z-base: 0; /* bg-illustration */ --z-content: 1; /* cards, scroll-body */ --z-elevated:2-3; /* match header elements, tab-bar */ --z-sticky: 50; /* profile sticky tabs */ --z-fixed: 99; /* bottom-nav, fab */ --z-chrome: 100; /* top header */
